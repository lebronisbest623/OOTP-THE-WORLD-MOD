# The World Mod

[한국어](./README.ko.md)

> Current Release Status: Beta

The World Mod is a custom OOTP 27 quickstart built around a broader global baseball universe.

This release includes a complete `THE_WORLD_MOD.quick` quickstart for OOTP 27, along with supporting resource files that help the world setup feel fuller, more polished, and more alive.

## Contents

- `THE_WORLD_MOD.quick` custom quickstart
- supporting visual resources for international leagues
- additional logos, uniforms, FaceGen, and ballpark assets
- expanded presentation for a richer world save experience

## Folder Structure

- `THE_WORLD_MOD.quick`: the custom quickstart
- `the-world-mod`: supporting resource files

## Installation

1. Copy `THE_WORLD_MOD.quick` into the OOTP 27 quickstart folder.
2. Copy the contents of `the-world-mod` into the appropriate OOTP 27 data folders if you want the included supporting resources as well.
3. If files with the same names already exist, back them up before replacing them.
